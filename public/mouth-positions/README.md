# Mouth Position Images

## Image Setup Instructions

When you have the mouth position images ready, place them in this folder (`/public/mouth-positions/`) with the following naming convention:

### Required Images:

1. `a-e-i.png` - For sounds: a, e, i
2. `b-m-p.png` - For sounds: b, m, p
3. `u.png` - For sound: u
4. `o.png` - For sound: o
5. `l.png` - For sound: l
6. `r.png` - For sound: r
7. `f-v.png` - For sounds: f, v
8. `th.png` - For sound: th
9. `w-q.png` - For sounds: w, q
10. `ch-j-sh.png` - For sounds: ch, j, sh
11. `general.png` - For general consonants (c, d, g, h, k, n, s, t, x, y, z)

## Image Specifications

- **Format**: PNG (with transparency preferred)
- **Size**: Recommended 400x400px or larger (square aspect ratio)
- **Background**: Transparent or white
- **Color**: Should match your brand colors (#152341, #EE8B44, #2A9896, #4ADEC0) if possible

## Current Status

Currently, the app shows a placeholder design. Once you add the images with the correct filenames, they will automatically appear in the Mouth Position Guide component.

## Testing

After adding images:
1. The component will automatically load images based on the sound being practiced
2. If an image is missing, the placeholder will show
3. The component maps sounds to the appropriate mouth position image
